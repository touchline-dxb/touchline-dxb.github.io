```

2. Add property: touchlinesport.ae
3. Verify ownership (HTML tag method recommended)
4. Go to Sitemaps → Add sitemap: sitemap.xml
5. Go to URL Inspection → Enter: https://www.touchlinesport.ae/careers
6. Click "Request Indexing"
```

## 🔄 Monthly Updates (First Monday of Month)

### 1. Refresh Job Posting Dates
Update in HTML (around line 1750):
```javascript
"datePosted": "2025-03-01",  // Update to current month
"validThrough": "2025-12-31"  // Keep end of year
```

### 2. Update Sitemap
Change lastmod date in sitemap.xml:

## 🆘 Troubleshooting

### If Page Not Indexed After 1 Week
1. Check robots.txt isn't blocking
2. Verify sitemap submitted
3. Request indexing again
4. Check for technical errors
5. Ensure page is live and accessible

### If No Rich Results Appearing
1. Validate structured data
2. Wait 2-3 weeks (Google needs time)
3. Check Search Console for errors
4. Ensure all required fields present

### If Low Click-Through Rate
1. Improve meta description
2. Add more compelling title
3. Test different descriptions
4. Add numbers/stats to title

### If High Bounce Rate
1. Improve page load speed
2. Make CTA more prominent
3. Simplify application process
4. Add more engaging content
